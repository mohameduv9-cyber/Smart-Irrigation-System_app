🌿 Smart Irrigation System
A cutting-edge solution designed to optimize water usage in agriculture and gardening through automation and real-time data analysis. This app allows users to monitor soil moisture and control irrigation systems remotely.

🚀 Features
Real-time Monitoring: Track soil moisture, temperature, and humidity levels.

Automated Watering: Smart scheduling based on sensor data to prevent over-watering.

Weather Integration: Syncs with local weather forecasts to skip irrigation on rainy days.

Remote Control: Manual override via the mobile/web interface from anywhere.

Data Analytics: Detailed charts showing water consumption patterns and plant health history.

🛠️ Tech Stack
Frontend: [Flutter ]

Backend: [Node.js / Firebase]

Hardware/IoT: [ESP32]

Database: [fire base]
